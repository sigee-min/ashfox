---
title: Guide
description: Blockbench Desktop 중심 제작 흐름을 위한 서술형 가이드.
---

이 가이드는 Blockbench에서 직접 에셋을 제작하는 사용자를 위한 흐름 문서입니다. 목적은 빠른 실행보다 작업 순서의 이유를 이해하는 것이며, 같은 품질로 반복 가능한 제작 루프를 만드는 데 있습니다.

먼저 플러그인 설치와 MCP 연결을 확인한 뒤, 모델 구조를 먼저 정리하고 텍스처 매핑과 페인팅으로 넘어가세요. 애니메이션은 구조와 표면 표현이 안정된 이후에 적용하는 것이 수정 비용을 줄입니다. 마지막에는 검증과 프리뷰를 거쳐 내보내기를 수행하면 결과 품질이 안정적입니다.

특정 작업을 즉시 처리해야 할 때는 작업 플레이북을 사용하면 되고, 전체 흐름을 설계할 때는 이 가이드와 가이드 허브를 먼저 읽는 것이 좋습니다.
