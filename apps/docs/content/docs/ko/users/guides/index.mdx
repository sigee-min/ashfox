이 페이지는 태스크 실행 전에 전체 맥락을 잡기 위한 서술형 가이드 모음입니다. 설치와 연결이 끝난 뒤 어떤 순서로 모델링, 텍스처, 애니메이션, 내보내기를 진행할지 판단할 때 먼저 읽는 것을 권장합니다.

단순 명령 순서보다 단계의 목적과 의사결정 기준을 설명하는 데 초점을 맞췄습니다. 작업이 급할 때는 플레이북으로 바로 내려가고, 품질과 재현성을 높이려면 아래 가이드 순서대로 읽으세요.
