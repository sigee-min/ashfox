텍스처 도구는 표면 표현과 연결된 작업을 담당합니다. 텍스처 리소스를 읽고, 대상에 바인딩하고, 페인팅 연산을 적용하며, 불필요한 텍스처를 정리할 수 있습니다.

텍스처 품질을 안정적으로 유지하려면 모델 구조 변경과 텍스처 변경을 분리하지 말고 가까운 단계에서 함께 관리하는 것이 좋습니다. 그래야 검증과 내보내기 결과가 작업 의도와 일치합니다.
