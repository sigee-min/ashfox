애니메이션 도구는 클립 생명주기와 키프레임 작성 흐름을 기준으로 구성됩니다. 클립을 생성·수정·삭제하고, 포즈와 트리거 키프레임을 타임라인에 배치해 재현 가능한 애니메이션을 만들 수 있습니다.

실무에서는 클립 식별자와 프레임 기준을 명확히 유지하는 것이 중요합니다. 포즈 채널과 트리거 채널을 분리해 관리하면 디버깅과 유지보수 효율이 크게 올라갑니다.
