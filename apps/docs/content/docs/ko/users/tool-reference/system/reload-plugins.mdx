## 개요
- 도구 이름: `reload_plugins`
- 카테고리: 시스템

## 도구 설명
Reloads Blockbench plugins (confirm required).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "confirm"
  ],
  "additionalProperties": false,
  "properties": {
    "confirm": {
      "type": "boolean"
    },
    "delayMs": {
      "type": "number"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `confirm` | `boolean` | ✅ | - | - |
| `delayMs` | `number` | ❌ | - | - |

## 최소 요청 예시
```json
{
  "confirm": true
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "scheduled": true,
    "delayMs": 0,
    "method": "devReload"
  }
}
```
