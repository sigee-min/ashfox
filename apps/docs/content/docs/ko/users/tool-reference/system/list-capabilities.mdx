## 개요
- 도구 이름: `list_capabilities`
- 카테고리: 시스템

## 도구 설명
Returns plugin capabilities and limits. Tool schemas are strict (extra fields are rejected).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false
}
```

## 필드 레퍼런스
입력 필드가 없습니다.

## 최소 요청 예시
```json
{}
```

## 응답 예시
```json
{
  "structuredContent": {
    "pluginVersion": "value",
    "blockbenchVersion": "value",
    "formats": [
      {
        "format": "Java Block/Item",
        "animations": false,
        "enabled": false
      }
    ],
    "limits": {
      "maxCubes": 0,
      "maxTextureSize": 0,
      "maxAnimationSeconds": 0
    }
  }
}
```
