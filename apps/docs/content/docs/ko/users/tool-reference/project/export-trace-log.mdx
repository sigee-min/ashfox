## 개요
- 도구 이름: `export_trace_log`
- 카테고리: 프로젝트

## 도구 설명
Flushes the trace log to disk (writeFile/export) and returns the resource URI for the in-memory log.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "mode": {
      "type": "string",
      "enum": [
        "auto",
        "writeFile",
        "export"
      ]
    },
    "destPath": {
      "type": "string"
    },
    "fileName": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `mode` | `enum("auto", "writeFile", "export")` | ❌ | - | enum: "auto", "writeFile", "export" |
| `destPath` | `string` | ❌ | - | - |
| `fileName` | `string` | ❌ | - | - |
