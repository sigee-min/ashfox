## 개요
- 도구 이름: `update_cube`
- 카테고리: 모델링

## 도구 설명
Updates a cube by id/name.

  "type": "object",

    },

      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,

    },
    "to": {

        "type": "number"
