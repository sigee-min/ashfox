---
title: 모델링 도구
description: 본, 큐브, 메시 기반 모델링 작업 도구.
---

모델링 도구는 구조를 만들고 수정하는 데 사용됩니다. 본, 큐브, 메시를 추가·수정·삭제하면서 계층과 지오메트리의 일관성을 유지하도록 설계되어 있습니다.

실무에서는 큰 변경을 한 번에 넣기보다 작은 단위로 나누어 적용하고 상태를 확인하는 방식이 안정적입니다. 이렇게 작업하면 revision 충돌이나 복구 비용을 줄이면서 반복 편집을 진행할 수 있습니다.
