## 개요
- 도구 이름: `add_bone`
- 카테고리: 모델링

## 도구 설명
Adds a bone to the current project.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "name"
  ],
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "parent": {
      "type": "string"
    },
    "parentId": {
      "type": "string"
    },
    "pivot": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "rotation": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "scale": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "visibility": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | - | - |
| `name` | `string` | ✅ | - | - |
| `parent` | `string` | ❌ | - | - |
| `parentId` | `string` | ❌ | - | - |
| `pivot` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `rotation` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `scale` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `visibility` | `boolean` | ❌ | - | - |
| `ifRevision` | `string` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `includeDiff` | `boolean` | ❌ | - | - |
| `diffDetail` | `enum("summary", "full")` | ❌ | - | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "name": "value"
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "id": "value",
    "name": "value"
  }
}
```
