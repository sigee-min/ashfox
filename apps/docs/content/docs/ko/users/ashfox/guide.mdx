title: Guide
description: Ashfox 서비스 운영 팀을 위한 서술형 운영 가이드.
---

이 가이드는 Ashfox 서비스를 운영하고 외부 MCP 클라이언트와 연동하는 팀을 위한 문서입니다. 핵심은 배포, 상태 점검, 클라이언트 연동을 같은 운영 모델로 관리해 장애 복구 시간을 줄이는 것입니다.

먼저 로컬 또는 스테이징 스택을 재현 가능하게 구성하고, 엔드포인트 연결성과 capabilities 동기화, revision 기반 변경 안전성을 확인하세요. 운영 단계에서는 검증, 프리뷰, 내보내기 확인을 명시적 체크포인트로 유지해야 품질 저하를 빠르게 차단할 수 있습니다.

긴급 작업이 필요할 때는 작업 플레이북이 빠르지만, 팀 온보딩이나 운영 설계에는 이 가이드가 우선입니다.
