---
title: Modeling Tools
description: Bone, cube, and mesh modeling operations.
---

Modeling tools are used to build and evolve structure. You can create, update, or remove bones, cubes, and meshes while preserving predictable hierarchy and geometry boundaries.

A common pattern is to apply structural changes in small batches, check state, and continue iteratively. This keeps revisions manageable and minimizes conflict when multiple agents or retries are involved.
