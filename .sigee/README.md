# SIGEE Governance

이 디렉터리는 Ashfox 저장소의 계획/오케스트레이션 거버넌스 자산을 보관합니다.

## 운영 원칙
- 제품 실행 계획은 `.sigee/product-truth/`를 단일 사실원(SSoT)으로 삼아 추적합니다.
- 실행 큐 및 임시 산출물은 기본적으로 `.sigee/.runtime/`에 생성됩니다.
- `.sigee/.runtime/`은 로컬 실행 상태이므로 버전관리 대상이 아닙니다.

## 주요 경로
- 정책: `.sigee/policies/`
- 제품 진실원: `.sigee/product-truth/`
- 런타임: `.sigee/.runtime/`
